Базовая cтруктура html страницы
    <html> Внутри него располагается вся информация. Благодаря этому тегу браузер понимает, где начинается контент, который необходимо обработать как HTML.
    • <head> Метаинформация, не показывается на странице Тег [head] означает заголовок html документа. В тегах head хранится информация для браузеров и поисковых систем.
      • <title> Название вкладки </title>
      • <meta Тег хранит метаданные, ссылки на файлы CSS, ссылки на иконки, данные о авторе и важные ключевые слова, описывающие документ. >
      </head>
    • <body> Основное тело HTML документа, где размещается весь контент. Этот элемент обязателен к использованию и используется на странице один раз.
      • <header> Шапка сайта. Может повторяться на других страницах. <header>
      • <main> Основной контент. Остается только на этой странице. <main>
      • <footer> Подвал сайта. Может повторяться на других страницах. <footer>
      </body>
    </html>
Базовые теги html страницы
  • <p> Тег параграф, это блочный элемент, всегда начинается с новой строки. Он означает абзац основного содержания html документа.<p>
  • <h1> h1, h2, h3, h4, h5, h6 – это шесть заголовков показывающих относительную важность блока после заголовка в зависимости от уровня заголовка <h1>
  • <br> Тег устанавливает перевод строки в том месте, где этот тег находится.одиночный тег, для перехода на другую строку
  • <a href="https://www.youtube.com/" target="_blank"> Ссылка </a> Обязательный атрибут тега <a> href указывает ссылку на веб-страницу
  • <strong> Выделить жирным <strong>
  • <em> Курсивный текст <em>
  • <input placeholder="поле поиска">
  • <table> Тег служит контейнером для элементов, определяющих содержимое таблицы.

  • <ul> Используется для неупорядоченного списка - в частности для маркированного списка.
            <li> элемент списка <li>
            <li> элемент списка <li>

    </ul>
  • <div>
            Элемент <div> является блочным элементом и предназначен для выделения фрагмента документа с целью изменения вида содержимого. Как правило, вид блока управляется с помощью стилей. Чтобы не описывать каждый раз стиль внутри тега, можно выделить стиль во внешнюю таблицу стилей, а для тега добавить атрибут class или id с именем селектора.
    </div>
